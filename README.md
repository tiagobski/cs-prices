# Intro

This project aims to collect skin prices from multiple sources.

Sources implemented:

- [x] CSFloat
- [x] GamerPay
- [x] Waxpeer
- [x] ShadowPay

# Getting started

1. Populate file `.env`
2. Execute `py csgo-api/_downloader.py`
3. Execute `py csgo-api/_gen_list_skin_names.py`
4. Configure `config.py`
5. Execute `py script.py`
